<ul>: 번호없는 리스트
<ol>: 번호있는 리스트
<dl>: 정의 리스트
<li>, <dt>, <dd>: 리스트 항목 표현
리스트 항목 안에도 테스트, 이미지, 링크, 다른 리스트를 넣을 수 있다.


리스트의 시작
<ul>
	<li> </li> 리스트 항목(list item)
	<li> </li>
	<li> </li>
</ul>
리스트의 끝

<dl> : 정의 리스트< definition list>, 항목들과 함께 항목들의 정의(설명)가 표시되는 리스트
	dt: definition term, 정의 항목
	dd: definition description, 정의 부가설

<a> : 텍스트나 이미지에 링크를 걸고, 링크를 클릭하면 연결된 다른 문서로 이동
- 밑줄, 글자색 등 자동으로 스타일이 지정
- 마우스를 올리면 손 모양 커서로 변경
- href 속성 : 링크의 목적지
- target 속성 : 링크 클릭 시, 새로운 페이지가 어디에 열리는 지를 지정

_self 기본적 클릭한 창/탭에서 새로운 페이지를 적재 (기존 창에서 링크)
_blank 새로운 창/탭에서 새로운 페이지를 적재 (새창에서 링크)
_parent 상위 프레임에서 새로운 페이지를 적재
_top 창 전체에서 새로운 페이지를 적재


id지정 : html요소의 고유 id지정. 공백을 포함하면 안됨
- 주의점 : 브라우저에서 id는 유일해야한다. 영어 소문자로 시작해야하며, 영 소문자, 숫자로만 이루어 지는것이 좋다.


<img> : 이미지를 웹페이지에 삽입할 때 사용, 이미지파일은 대부분 적용 가능하다.
- src : 이미지 파일 이름을 지정
- width : 이미지 가로 크기
- height : 이미지 세로 크기
- alt : 브라우저가 어떤 이유로 이미지를 화면에표시하지 못했을 경우에, 표시되는 대체 텍스트(alternate text), 웹접근성
	alt는 무조건 적어줘야 한다.

<img src = "asdf.jpg" width = "300" height = "230" alt = "image alternate text">
