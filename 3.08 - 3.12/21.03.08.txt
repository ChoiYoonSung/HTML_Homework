CSS

문서의 구조 및 내용 - HTML
문서의 스타일 - CSS
문서의 동작 - Java Script

HTML/CSS/Java Script 모두 순서로 인해 작동이 달라질 수 있음

장점
- 한 번에 여러 페이지의 레이아웃을 제어할 수 있음
- 별도의 css파일로 생성 가능(외부 스타일 시트)
- 개발 시간 단축 및 공유 간으
- 거대하고 복잡한 사이트를 관리 시 필요

인라인 > 내부 > 외부 순으로 우선순위를 띔

외부스타일시트
 - 스타일 시트를 외부에 파일로 저장하는 것
 - 많은 페이지에 동일한 스타일을 적용하려고 할 때 좋은 방법

내부CSS
- html파일 내에 style을 지정해주는 것

다중 스타일 시트
- 인라인 스타일 시트
- 헤드 섹션에 저장된 외부 스타일 시트와 내부 스타일 ㅣ트
- 웹 브라우저의 디폴트 값
- HTML의 헤드 섹션에 외부 CSS, 내부CSS 둘 다 기재되어 있는 경우,
  뒤에 선언된 CSS로 적용됨
- 공통적으로 사용되는 스타일은 <body>요소의 스타일에 정의하는 것이 편함

CSS3 문법
- 선택자{속성:값;}
- 끝에 반드시 ;을 적어준다

선택자
- HTML 요소를 선택하는 부분

자주 사용되는 선택자
- 요소
- 클래스 : .클래스
- 아이디 : #아이디
- 전체 (페이지 전체) : *
- 속성 : 
- 의사 : 선택자의 활동(이벤트와 연관)
- 그룹 : ,를 통해 여러 요소들 공통
- 자손, 자식 : 부모 자식 (하위 자식들에게도 포함) 
	  or 부모 > 자식 (부모 선택자의 직계 자식의 속성만)
